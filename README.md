# Penelcom Next

Fresh Next.js, TypeScript, and Tailwind CSS rebuild of the Penelcom website.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Local static assets in `public/assets`

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```
